# 🍯 Honey Community (꿀사모 커뮤니티)

간단한 웹 기반 커뮤니티 서비스로, 게시판 기능과 댓글 기능, 그리고 챗봇을 포함한 웹 애플리케이션입니다.
Flask를 기반으로 구현되었으며, JSON 파일을 이용하여 데이터를 저장합니다.

---

## 📌 프로젝트 개요

본 프로젝트는 사용자들이 자유롭게 글을 작성하고 소통할 수 있는 커뮤니티 웹 서비스를 구현하는 것을 목표로 합니다.
또한 향후 AI 기반 허니팟 시스템으로 확장하기 위한 기반 구조를 포함하고 있습니다.

---

## 🛠️ 사용 기술

* Python (Flask)
* HTML / CSS
* JSON (데이터 저장)
* Jinja2 (템플릿 엔진)

---

## ✨ 주요 기능

### 🔐 로그인 기능

* 간단한 인증 (admin / 1234)
* 로그인 성공 시 게시판 이동

### 📝 게시판 기능

* 글 작성 / 수정 / 삭제 (CRUD)
* 게시글 목록 출력
* 검색 기능 (제목 기반)

### 💬 댓글 기능

* 게시글별 댓글 작성
* 댓글 목록 출력

### 🤖 챗봇 기능

* 세션 기반 대화 유지
* 사용자 입력 및 응답 처리
* 향후 AI API 연동 가능 구조

### 💾 데이터 저장

* JSON 파일(posts.json)을 활용하여 데이터 유지
* 서버 재시작 시에도 데이터 유지 가능

## 🚀 실행 방법

1. Python 설치
2. 필요한 라이브러리 설치

```
pip install flask
```

3. 서버 실행

```
python app.py
```

4. 브라우저 접속

```
http://127.0.0.1:5000
```

## ⚠️ 참고 사항

* 데이터는 JSON 파일로 저장되며, 별도의 데이터베이스는 사용하지 않습니다.
* OpenAI API 연동 기능은 현재 비용 문제로 비활성화되어 있으며, 추후 확장 가능합니다.
